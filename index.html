qry.js

The DOM library where $() returns the real element — no wrapper to unwrap, fully chainable.

No new object model to learn: you work with the element you already know, and qry.js just makes it shorter.

A guided tour

One concept per card — code on top, the live result below

From the basics to traversal and a real chart. Every card is wired with qry.js itself; click to run it.

.cls() — add / remove / toggle / check

$('#box').cls('~active')  // + add  - remove  ~ toggle  ? check
target

One prefixed string does what classList needs four calls for.

.text() .html() — get or set content

$('#out').text('safe');  // escaped
$('#out').html('<b>rich</b>');  // parsed
original content

.text() escapes, .html() parses — same method shape, get with no argument.

.on() — listen for events

$('#inc').on('click', () => $('#count').text(++n));
0

.on() is addEventListener, chainable — pass an arrow function, no this surprises.

$.create() .mount() — build & insert

$.create('span', { class: 'pill', text }).mount('#sink');

Create with props in one call, then .mount() it into any target.

.delegate() — one listener for many items

$('#list').delegate('li', 'click', function () {
  this.cls('~done');  // `this` = the clicked <li>
});
  • click me
  • then me
  • toggle done

New items work instantly — the listener lives on the parent, not each child.

Traversal — move around the DOM

cell.prev()  cell.next()  cell.idx()  // neighbours & position

Click a cell — its neighbours light up via .prev() / .next().

$.all() — a real Array, not a wrapper

$.all('.dot').forEach((d, i) => …)  // native .forEach / .map / .filter

No special collection API to learn — it's a plain Array, so array methods just work.

$.tip() — shared tooltip, follows the cursor

$.tip(swatch, name, hex);

One call binds enter / move / leave. Hover the swatches.

$.scale() — map a value to a range

$.scale(v, 0, max, 8, 110)  // value → bar height (px)

The one math helper a chart needs — bars built from plain elements, each with a $.tip.

.val() .serialize() — read a form

$('#form').serialize().toString()  // → URLSearchParams
name=Ada&city=Fribourg

Null-safe — a miss never throws

$('#ghost').text('x');  // warns, returns a node, no crash

Open the console, then click.

Install

Drop in one file. Start writing $.

qry.js defines the global $ — a classic script, no bundler, no import.

<!-- local -->
<script src="qry.js"></script>

<!-- pinned to a release (recommended) -->
<script src="https://cdn.jsdelivr.net/gh/Bloechle/qry@1.1.0/qry.js"></script>

Then anywhere: $('#app').cls('+ready'). Full API in the README.
